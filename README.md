# Sign in & Sign up

> Implement account management functions.

## Background
The [Okta](https://www.okta.com/) provides API for developers to build identity controls into applications, and I'm trying implemet account manage function by this API.

## Features
- Register
- Sign in 
- Show User Profile
- Modify User Profile
- Auto Login
